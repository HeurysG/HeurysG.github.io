Cryptocurrency Forecasting

What problem I was curious about

Crypto forecasts are everywhere, but most ignore uncertainty, risk tolerance, or usability. I wanted a system that forecasts price trends and helps users think about risk in practical terms.

What I did

Built a full forecasting pipeline with multiple time-series models (baselines, ARIMA, Prophet) and wrapped it in a simple app so users can explore forecasts and portfolio allocations by risk profile.

What I found

Assets behaved very differently under the same assumptionsâ€”some with stable trends, others dominated by volatility and regime shifts. Forecasts paired with risk context proved far more useful than raw predictions alone.

Why it matters

  • Work with real financial time-series data.
  • Compare and reason about multiple forecasting approaches.
  • Build analytics meant to be used, not just analyzed.

Quick install (from README)

  1. Download data to data folder (link in README).
  2. Backend: cd api && pip install -r requirements.txt && python server.py
  3. Frontend: cd client && npm install && npm start
  4. Open http://localhost:3000

Methods

Forecasting with baselines, Prophet, and ARIMA; risk-aware portfolio suggestions; scenario analysis by market cap and risk tolerance.

Highlights

  • Interactive portfolio insights and forecast visualizations.
  • Personalized risk profiles (conservative, moderate, aggressive).
  • End-to-end pipeline with React frontend and Python API backend.
