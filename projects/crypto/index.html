Cryptocurrency Forecasting

Forecasting crypto prices with a full-stack TradeCrypto app. Backend APIs feed ML forecasts and portfolio insights; the React client visualizes performance and suggestions.

Quick install (from README)

  1. Download data to data folder (link in README).
  2. Backend: cd api && pip install -r requirements.txt && python server.py
  3. Frontend: cd client && npm install && npm start
  4. Open http://localhost:3000

Methods

Forecasting with baselines, Prophet, and ARIMA; risk-aware portfolio suggestions; scenario analysis by market cap and risk tolerance.

Highlights

  • Interactive portfolio insights and forecast visualizations.
  • Personalized risk profiles (conservative, moderate, aggressive).
  • End-to-end pipeline with React frontend and Python API backend.
